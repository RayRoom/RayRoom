Examples
========

This section provides a brief overview of the examples available in the `examples/` directory of the repository.

.. contents::
   :local:

Ray Tracing Demo
----------------

- **File**: `demo_raytracing.py`
- **Description**: Demonstrates a basic simulation using the ray tracing engine.

Hybrid Demo
-----------

- **File**: `demo_hybrid.py`
- **Description**: Shows how to use the hybrid engine, combining ISM for early reflections and ray tracing for late reverberation.

Radiosity Demo
--------------

- **File**: `demo_radiosity.py`
- **Description**: An example of using the radiosity engine for modeling diffuse sound fields.

Spectral Demo
-------------

- **File**: `demo_spectral.py`
- **Description**: Demonstrates the spectral engine, which uses FDTD for low frequencies and a geometric approach for high frequencies.

DGFEM Demo
----------

- **File**: `demo_dgfem.py`
- **Description**: An example of using the Discontinuous Galerkin Finite Element Method (DGFEM) engine.

Polygon Room Demo
-----------------

- **File**: `demo_polygon_room.py`
- **Description**: Shows how to create and simulate a room with a complex polygonal floor plan.

For more details, please see the `README.md` file in the `examples/` directory.
